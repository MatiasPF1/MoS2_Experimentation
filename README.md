# First Functional Version
- Work For The Advanced Quantumn Materials Lab, it aims to facilitate and have a general hub of the following:<br>
             -Generate Personalized XYZ and Params FIles of MoS2 <br>
             -Generate Scanning Transmission Electron Microscopy(STEM) Images<br>
             -Post-processing of simulated STEM images to inject realistic noise profiles<br>
             -Res-Unet Based Classification of Vacancies and Polymorphs
